## ğŸ†˜ Support & Troubleshooting

### Common Issues

**Can't login to admin?**
- Clear browser cookies: Settings â†’ Privacy â†’ Clear Browsing Data
- Try different browser
- Contact: support@himalayan-ai-tech-pro.com

**Payment not processing?**
- Verify merchant account is active (check Khalti/eSewa dashboard)
- Check webhook configuration
- Test payment in sandbox mode first
- Contact payment gateway support

**Emails not arriving?**
- Check spam folder
- Verify email in your profile
- Enable email notifications in dashboard

**Page loading slowly?**
- Check internet connection
- Clear browser cache (Ctrl+Shift+Delete)
- Try incognito/private mode
- Contact support if issue persists
